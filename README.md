# 爆款标题公式库

> 6大类标题公式合集，覆盖主流平台标题写作需求

---

## Features / 功能特点

| 功能 | 说明 |
|------|------|
| 公式分类 | 数字型/对比型/悬念型/痛点型/情绪型/反常识型 |
| 平台适配 | 公众号/小红书/知乎/抖音/头条 等主流平台推荐 |
| 结构化数据 | JSON格式公式库，含公式+案例+适用场景 |
| 可读版本 | Markdown格式分类索引，便于查阅 |
| 持续更新 | 欢迎贡献新公式，持续扩充库内容 |

## Installation / 安装

无需安装，直接查阅 Markdown 文件或使用 JSON 数据。

```bash
# 克隆仓库
git clone https://github.com/yourusername/viral-title-formula-library.git

cd viral-title-formula-library

# 查看公式库
cat 标题公式库.md
```

## Usage / 使用方法

### 基础用法

1. 找到匹配的公式类型
2. 填入你的内容关键词
3. 微调适配你的风格

### 公式分类一览

| 分类 | 公式模板数 | 适用场景 | 示例 |
|------|:--------:|----------|------|
| 数字型 | 5 | 数据驱动、清单类 | `{数字}个{领域}必备{技能}` |
| 对比型 | 4 | 冲突制造、差异强调 | `A vs B：哪个更适合{场景}` |
| 悬念型 | 4 | 好奇心激发 | `{行业}里藏着{秘密}` |
| 痛点型 | 4 | 问题解决、刚需 | `还在{错误做法}？{正确方法}来了` |
| 情绪型 | 4 | 共情、共鸣 | `当{场景}发生时，我{情绪}` |
| 反常识型 | 4 | 认知颠覆 | `你以为{常识}，其实是{真相}` |

### 平台适配建议

| 平台 | 推荐公式类型 | 字数建议 | 特点 |
|------|-------------|:--------:|------|
| 公众号 | 痛点型+情绪型 | 15-25字 | 粉丝粘性强，深度内容 |
| 小红书 | 数字型+对比型 | 10-15字 | 搜索流量为主，图文结合 |
| 知乎 | 反常识型+数字型 | 20-30字 | 专业性强，问答形式 |
| 抖音 | 悬念型+情绪型 | 8-12字 | 快节奏，强钩子 |
| 头条 | 数字型+痛点型 | 15-20字 | 信息流分发，热点敏感 |

### JSON 数据结构

```json
{
  "category": "数字型",
  "formula": "{数字}个{领域}必备{技能}",
  "example": "10个职场必备沟通技巧",
  "platform": ["公众号", "小红书", "知乎"],
  "emotion": "实用",
  "difficulty": "easy"
}
```
