# 爆款标题公式库

写了10篇公众号，阅读量还是两位数？
**用20+爆款标题公式，一键生成高点击率标题。**

[![HTML](https://img.shields.io/badge/打开即用-无需安装-orange)]()

## 老板看了都说好

- **标题公式库**：20+经典爆款公式，按场景分类（情感/干货/争议/悬念）
- **一键生成**：选择公式 → 填参数 → 秒出标题
- **效果追踪**：统计热门公式，帮你找到自己的"爆款密码"
- **导出分析**：Markdown格式，方便复盘

## 怎么用

打开 `在线生成器.html` → 选公式 → 填内容 → 复制标题
